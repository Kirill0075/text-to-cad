# Mechanism Source Index

This folder contains mechanism CAD/SDF fixtures sourced from the YouTube
channel [thang010146](https://www.youtube.com/@thang010146/videos). Original
mechanism design, video animation, and downloadable STEP/Inventor source files
are credited to the author/channel `thang010146`.

Each sourced mechanism directory exposes a canonical `<folder>.step` assembly
and `<folder>.sdf` visualization model. The downloaded archives from the video
descriptions are kept under each mechanism's `assets/` directory as
`source_step.zip` and `source_inventor.zip`, with extracted source files in the
hidden `.source_step_files/` and `.source_inventor_files/` directories.
