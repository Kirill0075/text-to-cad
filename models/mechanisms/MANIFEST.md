# Mechanism STEP Assembly Downloads

Source channel: [thang010146](https://www.youtube.com/@thang010146/videos)

Each folder exposes a top-level `<folder>.step` and `<folder>.sdf`. Original
downloads live under `assets/`:

- `assets/source_step.zip` and `assets/.source_step_files/`
- `assets/source_inventor.zip` and `assets/.source_inventor_files/`
- `assets/gen_sdf.py`
- `assets/.sdf_meshes/`

The generated SDFs use the shared `sdf_common.py` generator and render as static
model structure in CAD Explorer.
