# thang010146 Demo STEP Assets

This folder is a flattened CAD Explorer demo bundle for mechanism assemblies
inspired by the YouTube channel
[thang010146](https://www.youtube.com/@thang010146/videos). Original
mechanism design, video animation, and downloadable source files are credited
to the author/channel `thang010146`.

Only directory-level files are kept here for the demo:

- `*.step`: canonical mechanism assemblies.
- `.*.step.glb`: generated render/selector artifacts paired with the STEP.
- `.*.step.js`: optional interaction sidecars paired with the STEP.

The extracted per-mechanism source archives, Inventor files, SDF meshes, and
generation scripts are intentionally omitted from this demo bundle. Source
links remain below for provenance.
